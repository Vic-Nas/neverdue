How to set up forwarding

✉️ Gmail
  1. Open gmail.com and go to Settings → See all settings
  2. Click the Filters and Blocked Addresses tab
  3. Click Create a new filter
  4. Leave fields blank to forward all, or fill in a sender to filter
  5. Click Create filter then check Forward it to
  6. Add {{ request.user.username }}@user.{{ DOMAIN }} as the forwarding address
  7. Confirm the verification email Google sends to your NeverDue inbox
📘 Outlook
  1. Open outlook.com and go to Settings → Mail → Rules
  2. Click Add new rule
  3. Set condition: Apply to all messages (or pick a sender)
  4. Set action: Forward to {{ request.user.username }}@user.{{ DOMAIN }}
  5. Save the rule
🍎 Apple Mail / iCloud
  1. Go to icloud.com/mail
  2. Click the Settings gear → Rules
  3. Click Add a Rule
  4. Set condition and action to Forward to {{ request.user.username }}@user.{{ DOMAIN }}
  5. Save
📧 Yahoo Mail
  1. Go to mail.yahoo.com → Settings → More settings → Filters
  2. Click Add new filters
  3. Set the filter and choose Forward to
  4. Enter {{ request.user.username }}@user.{{ DOMAIN }}
  5. Save
